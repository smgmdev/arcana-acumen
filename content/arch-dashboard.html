Threat dashboard specifications

Three integrated panels: live threat map, threat intelligence, and alert & notification system — in the local language and English.

Panels

Real-time threat map

  • Region of interest + adversary territory
  • Threat objects plotted with trail
  • Color coded: Red / Amber / Green
  • Satellite imagery underlay
  • AIS ship positions overlaid
  • Flight tracks overlaid
  • Drone detections from ground sensors
  • Local language + English toggle

Threat intelligence

  • Active threats with confidence %
  • Predicted trajectory visualization
  • Time-to-impact countdown
  • AI-generated narrative (AR/EN)
  • Historical pattern match display
  • OSINT signals that triggered alert
  • Analyst annotation capability
  • One-click PDF briefing export

Alerts & notifications

  • Tiered: Info / Warning / Critical
  • Push to secure mobile app
  • SMS to command roster
  • Secure radio integration option
  • Escalation tree configurable
  • Alert acknowledgment tracking
  • False-positive feedback button
  • Full immutable audit log

# Commander dashboard — real-time WebSocket data flow

@app.websocket("/ws/threat-feed/{client_id}")
async def threat_feed(websocket: WebSocket, client_id: str):
    await manager.connect(websocket, client_id)
    try:
        async for event in kafka_consumer.subscribe("threats.processed"):
            threat  = ThreatEvent.parse(event)
            payload = {
                "type": "THREAT_UPDATE",
                "threat_id": threat.id,
                "alert_level": threat.alert_level,
                "lat": threat.lat, "lon": threat.lon,
                "trajectory": threat.projected_path,
                "impact_time_seconds": threat.time_to_impact,
                "confidence": threat.confidence,
                "narrative_local": threat.briefing_local,
                "narrative_en": threat.briefing_english,
                "sources": threat.contributing_sensors
            }
            await websocket.send_json(payload)
    except WebSocketDisconnect:
        manager.disconnect(client_id)
