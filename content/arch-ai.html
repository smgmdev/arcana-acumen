Architecture/AI pipeline

AI analysis pipeline

Three model stacks — computer vision, language AI, and prediction — orchestrated into one unified threat-intelligence output.

Model stacks

Computer vision

  • YOLOv9 base — real-time detection
  • Fine-tuned on adversary vehicle types
  • Fine-tuned on adversary UAV signatures
  • Adversary naval vessel classifier
  • Missile launcher shape detection
  • Change detection: before/after SAR
  • NVIDIA A100 GPU cluster
  • <500 ms per satellite tile

Language AI

  • Claude via Azure Local Gov Region
  • Local language–English translation
  • Open-source reporting monitoring
  • Entity extraction: people, places, units
  • Threat-relevance scoring
  • Auto briefing generation
  • Adversary state-media analysis
  • Data stays in local government Azure region

Prediction

  • LSTM time-series pattern matching
  • Historical pre-attack signature DB
  • Multi-domain signal correlation
  • Bayesian threat-probability engine
  • Outputs: probability, confidence, window
  • Retrained weekly
  • False-positive feedback loop
  • Alert threshold configurable

# AI threat analysis pipeline

async def analyze_threat_context(
    threat_obj: ThreatObject,
    satellite_context: SatelliteScene,
    osint_feed: OSINTBuffer
) -> ThreatIntelligence:

    # Step 1: Computer vision on satellite imagery
    cv_findings = await classify_satellite_objects(satellite_context)

    # Step 2: Language AI — local-language OSINT analysis (Claude via Azure Local Gov)
    osint_analysis = await claude_client.messages.create(
        model="claude-sonnet-4-6",
        system=ARCANA_MACE_SYSTEM_PROMPT_AR,
        messages=[{
            "role": "user",
            "content": f"Analyze for threat indicators:\n{osint_feed.recent_content}"
        }]
    )

    # Step 3: Multi-domain correlation + prediction score
    prediction = threat_model.predict(
        cv_findings=cv_findings,
        osint_signals=osint_analysis,
        sensor_data=threat_obj,
        historical_patterns=ATTACK_PATTERN_DB
    )

    # Step 4: Generate commander briefing in local language + English
    briefing = await generate_briefing(
        threat=threat_obj,
        analysis=prediction,
        language=["ar", "en"]
    )

    return ThreatIntelligence(
        threat=threat_obj,
        cv_findings=cv_findings,
        prediction=prediction,
        briefing=briefing,
        alert_level=compute_alert_level(prediction.probability)
    )
