Performance & security

Performance SLA

  • Drone detection alert: <500 ms end-to-end
  • Satellite analysis: <5 min from image acquisition
  • OSINT alert: <2 min from content publish
  • Dashboard refresh: real-time WebSocket
  • Targeting data update rate: 100 ms
  • System uptime SLA: 99.95%

Security

  • All data encrypted at rest (AES-256)
  • All data encrypted in transit (TLS 1.3)
  • Zero-trust network architecture
  • MFA required for all dashboard access
  • Full immutable audit log
  • Penetration testing: quarterly
