Architecture/Interception API

Targeting data output API

Arcana Mace outputs real-time targeting data via secure API to connected interception systems. Hardware-agnostic — the same targeting data feeds laser systems, electronic jamming, or existing Patriot / THAAD fire control.

Kill-chain flow

Detect

Threat confirmed by fusion engine

Multi-sensor fusion confirms threat. Confidence above threshold. ThreatObject created with position, velocity, heading, class.

Track

Real-time trajectory projection

Kalman filter tracks object position. Trajectory projected at 100ms intervals. Impact point predicted with ±50m accuracy at 10km range.

Authorize

Engagement authorization · human in the loop

Commander receives alert. One-button authorization on secure dashboard. Engagement authority always remains with human commander — AI never fires autonomously.

Output

Targeting data transmitted to interception system

Secure encrypted API sends real-time targeting coordinates, velocity vector, predicted intercept point, and recommended engagement window.

Targeting API

# Targeting data output — interception system API

class InterceptionTargetingAPI:

    def __init__(self, system: InterceptionSystem):
        self.system = system   # IronBeam | RaytheonHEL | Patriot
        self.authorized = False

    async def stream_targeting_data(
        self,
        threat: ThreatObject,
        commander_authorization: AuthToken
    ):
        # MANDATORY: human commander must authorize before streaming
        if not verify_authorization(commander_authorization):
            raise UnauthorizedEngagement("Commander auth required")

        async for position_update in self.track_threat(threat):
            intercept_solution = self.compute_intercept_point(
                current_pos=position_update,
                laser_range=self.system.max_range_m,
                engagement_time_ms=self.system.target_acquisition_ms
            )
            await self.system.send_targeting(TargetingPacket(
                azimuth=intercept_solution.azimuth,
                elevation=intercept_solution.elevation,
                range_m=intercept_solution.range,
                velocity_vector=position_update.velocity,
                time_to_intercept_ms=intercept_solution.tti,
                priority="HIGH"
            ))
Human-in-the-loop policy

Arcana Mace systems are designed as decision-support tools. No interception is initiated without explicit commander authorization. AI provides targeting solutions; humans authorize engagement. This is a hard architectural constraint, not a configurable setting.
