Capabilities/Automation

Automation

Arcana Mace has an integrated automatic guidance and recommendation system to improve analytical thinking. Users create modules, state the desired outcome, and AI runs the modules — then builds additional modules to close the gap to that goal.

Goal-driven module generation

The user defines the outcome they want. AI reviews the existing modules, runs them, identifies what is missing, and creates new modules automatically — each one designed to move the mission closer to the desired result. The operator remains in control; AI accelerates the path, it doesn't replace the decision-maker.

Input

User states desired output

The operator describes the goal — mission outcome, production target, strategic objective — in their own terms.

Analyse

AI runs current modules

Existing modules are executed against the current state of the platform; results and gaps are surfaced.

Generate

AI builds additional modules

Missing steps are generated as new modules and recommendations — each aimed at closing a specific gap to the goal.

Iterate

Continuous refinement

The loop repeats — new outcomes are proposed, new modules generated, and the operator approves, edits, or rejects each step.

Where automation applies

Mission strategy

Define a mission outcome — AI proposes a sequence of intelligence, deployment, and engagement modules to reach it.

Production

State a production target or supply goal — AI recommends factory allocation, product prioritisation, and dispatch routing.

Procurement

Define a capability gap — AI identifies the procurement order, transport mode, and timing required to close it.

Any module-driven workflow

The same automation pattern applies wherever modules can be composed — analytics, planning, simulation, logistics.

Analytical thinking, amplified

Automation doesn't replace the operator's judgement — it surfaces what's missing, accelerates the thinking, and proposes concrete next modules. The commander decides which to run.
